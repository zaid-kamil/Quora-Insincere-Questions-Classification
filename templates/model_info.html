Model Information

Algorithms used while Building the Model are:

  1. Naive Bayes
  2. Logistic Regression
  3. Support Vector Machine

Features

  1. Length : It is an Integer Value representing the Length of a Question

  2. Total Words: It is an Integer Value Representing the Count of Total Words in the Question

  3. Total StopWords: It is an Integer Value Representing the Count of Total StopWords in the Question.StopWords are commonly used words such as "the","is" "an" etc.

  4. Uppercase Count: It is an Integer Value Representing the Count of UpperCase letters in the Question

  5. BadWords Count: It is an Integer Value Representing the Count of Total Toxic or Bad Words in the Question.

  6. BadWords Ratio: It is a Float Value Representing the Ratio of bad Words to Total Words in the Question

  7. Punctuations Count: It is an Integer Value Representing the Count of Total Punctuation Characters in the Question

  8. Unique Words Count: It is an Integer Value Representing the Count of Total Unique Words in the Question

  9. Unique Words Ratio: It is a Float Value Representing the ratio of Unique Words to Total Words in the Question

Quora Insincere Questions Classification

QIQS is a ML web App to test the sincerity of a valid Quora Question.One can validate whether the given question is Sincere or Insincere based on the policies and terms and conditions of the Quora.
